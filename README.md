## **Endpoints**

| REST Method |      Endpoint      | Description                          |
| :---------: | :----------------: | :----------------------------------- |
|   **GET**   |     api/users      | view list of users                   |
|   **GET**   |   api/users/:id    | get user by id                       |
|  **POST**   |     api/users/     | add new user                         |
|   **PUT**   |   api/users/:id    | update user by id                    |
| **DELETE**  |   api/users/:id    | delete user by id                    |
